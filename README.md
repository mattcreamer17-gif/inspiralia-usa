# Inspiralia USA landing page

A single-purpose briefing page for Inspiralia USA, Inc., prepared for eMerge Americas 2026, Booth 510. The design uses Inspiralia's own brand pack (primary cyan-blue #1FADDD, Roboto Condensed headings, Roboto Flex body, light-surface treatment) so the page reads as native to inspiralia.us rather than as a vendor pitch, with Eduba credited only in the footer. The hero carries the fit-analysis headline verbatim, the body reflows section 7 of the fit analysis, and a $520M / 980 projects / 100+ experts / 20-year metric strip mirrors the rhythm of their own marketing. The case study is VigilOre (structural match on a document-heavy, expert-bottlenecked workflow). The CTA routes to Matt Creamer (CRO) via his 30-minute Calendly. All pricing, retainer, and revenue-share language has been stripped; only Inspiralia's own public metrics appear.
